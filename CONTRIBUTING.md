# Contributing to Termiaxial

Thank you for contributing! 🎉

## Quick Start

```bash
git clone git@github.com:angga30/termiaxial.git
cd termiaxial
npm install
npm run tauri dev
```

## Reporting Issues

- Check existing issues first
- Include: OS, steps to reproduce, expected vs actual behavior

## Pull Requests

1. Fork and create branch: `git checkout -b feat/your-feature`
2. Follow code style: `cargo fmt` and `npm run lint`
3. Test: `cargo test`
4. Commit: Use conventional commits (`feat:`, `fix:`, `docs:`)
5. Create PR with clear description

## Code Style

- **Rust**: `rustfmt`
- **TypeScript**: ESLint
- **Commits**: `feat:`, `fix:`, `docs:`, `refactor:`
